# Audio_classifier_

I had created two dataset one contain normal engine audio and other contain the audios of a faulty engine. 

Normal audio imported from kaggle at following link : "https://www.kaggle.com/datasets/malakragaie/car-diagnostics-dataset".
Abnormal audio imported from kaggle at following link : "https://www.kaggle.com/datasets/zeyadzsm/engine-sounds" .
